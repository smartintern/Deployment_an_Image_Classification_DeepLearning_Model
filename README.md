# Deployment an Image Classiffication Deep Learning Model

## Purpose and Scope
In this project, our goal is to classify snapshots from the camera server. We used a sample dataset for this. Our project involves training the model and finding which class the incoming images belong to. Thus, it tells us which picture belongs to which class
